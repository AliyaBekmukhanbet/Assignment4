# Assignment4

Контрольные вопросы к Assignment 4

1. В чём заключается отличие гибридных вычислений от вычислений только на
CPU или только на GPU?
Гибридные вычисления используют и CPU, и GPU совместно; CPU — для управления и логики, GPU — для массового параллелизма. Только CPU или только GPU используют один тип ресурсов.
2. Для каких типов задач целесообразно распределять вычисления между CPU и
GPU?
Для задач с тяжёлыми параллельными вычислениями и сложной логикой: численные расчёты, обработка изображений, машинное обучение.
3. В чём разница между синхронной и асинхронной передачей данных между CPU
и GPU?
Синхронная передача блокирует выполнение до завершения копирования, асинхронная — позволяет выполнять вычисления параллельно с передачей данных.
4. Почему асинхронная передача данных может повысить производительность
программы?
Потому что позволяет перекрывать передачу данных и вычисления, уменьшая простой CPU и GPU.
5. Какие основные функции MPI используются для распределения и сбора данных
между процессами?
MPI_Send, MPI_Recv, MPI_Bcast, MPI_Scatter, MPI_Gather, MPI_Reduce.
6. Как количество процессов MPI влияет на время выполнения программы и
почему?
Увеличение числа процессов снижает время вычислений, но повышает накладные расходы на обмен данными и синхронизацию.
7. Какие факторы ограничивают масштабируемость распределённых
параллельных программ?
Задержки сети, накладные расходы коммуникаций, несбалансированная нагрузка, ограничение памяти.
8. В каких случаях использование распределённых вычислений оправдано, а в
каких — неэффективно?
Оправдано для больших вычислительно ёмких задач; неэффективно для малых задач с частыми обменами данными.
